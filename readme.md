## Usage

- Currently the following sub commands are available
  - `worker`
  - `manager`
  - `run`
  - `status`
  - `stop`
  - `node`
- To learn how to use each type the subcommand name followed by `-h` or `--help` for details
  - Example: `go run main.go worker -h`
